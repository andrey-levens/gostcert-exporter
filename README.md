# gostcert-exporter

Prometheus-экспортер для мониторинга сроков действия открытых и закрытых сертификатов (OID 2.5.29.16), включая контейнеры.

Подробно:
- Сканирует папку (где находится скрипт) на наличие файлов сертификатов и контейнеров в форматах: `.crt`, `.pem`, `.cer`, `.der`, `.p7b`, `.p7c`
- Экспортирует метки Prometheus:
  - `cert_public_key_valid_from` — начало действия открытого ключа
  - `cert_public_key_valid_to`   — окончание действия открытого ключа
  - `cert_private_key_valid_from` — начало действия закрытого ключа (OID 2.5.29.16)
  - `cert_private_key_valid_to`   — окончание действия закрытого ключа (OID 2.5.29.16)
- Корректно работает с сертификатами, где срок действия закрытого ключа отличается от открытого

В российских сертификатах часто срок действия закрытого ключа (Private Key Usage Period) короче, чем у открытого сертификата.  
Если следить только за `not_valid_after` открытого ключа — можно пропустить просрочку закрытого ключа и получить неработающие подписи/шифрование задолго до официального окончания сертификата.

gostcert-exporter решает эту проблему и даёт настоящую картину по срокам жизни ключей.

Запустить можно как в Docker-контейнере, так и локально на машине.
